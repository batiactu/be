Pour windows :

- Installer cygwin (console ssh)
(modifier en fonction du chemin!))
- cd c:
- cd var/www/folderproject/tools/windows
- sh minifyalljscygwin.sh
=> le fichier /folderproject/js/cipallmin.js est régénéré!
=> le fichier /folderproject/js/allpluginsmin.js est régénéré!
mais il faut l'ouvrir et passer en utf8 le contenu (pspad => sélectionner le contenu puis Format>UTF-8)
